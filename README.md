# Tic Tac Toe with AI (Godot)

A Tic Tac Toe (3-in-a-row) game developed in Godot featuring a single-player mode against an AI opponent powered by the Minimax algorithm with Alpha-Beta pruning.

---

## About

This project is based on the following tutorial repository:
https://github.com/russs123/tictactoe_tut

The original project has been extended by implementing an AI opponent capable of playing optimally using Minimax with Alpha-Beta pruning.

## AI Implementation

The AI uses the Minimax algorithm to evaluate all possible game states and choose the optimal move.

Alpha-Beta pruning is applied to:

* Reduce the number of evaluated nodes
* Improve performance
* Maintain optimal decision-making

This results in an AI that cannot be beaten if played correctly.
